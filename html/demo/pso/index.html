PlanarSolarSystemApplet

A demonstration of JAutoDiff (A pure Java automatic differentiation library).
This applet program shows a dynamics of Sun-Earth-Moon system defined as

(Eq. 1)
xsun = 0
xearth = xsun + aearth (cos(θ0 earth + wearth(t - t0)), sin(θ0 earth + wearth(t - t0)))T
xmoon = xearth + amoon (cos(θ0 moon + wmoon(t - t0)), sin(θ0 moon + wmoon(t - t0)))T
where x denotes position of the astronomical objects.

Press the key 'v' on the applet, then you can see the dynamics in state spaces of velocity.
While only the above equations (Eq.1) are coded manually in this program,
JAutoDiff generates the derivative below (Eq.2) of (Eq.1) with respect to time t.

(Eq. 2)
vsun = 0
vearth = vsun + aearthwearth (-sin(θ0 earth + wearth(t - t0)), cos(θ0 earth + wearth(t - t0)))T
vmoon = vearth + amoonwmoon (-sin(θ0 moon + wmoon(t - t0)), cos(θ0 moon + wmoon(t - t0)))T

Actually, this program simulates the dynamics with initial condition such that
θ0 earth=θ0 moon=0 at an arbitrary time in the time interval t0 = [-50000.0, 50000.0] ⊂ R.
The state variables x, v are computed as real interval vectors using interval arithmetics library ia_math.
The rectangles in the applet show bounds of the state variables.

  • t : Change mouse gesture mode into [Translation] of view.
  • z : Change mouse gesture mode into [Zoom] of view.
  • p : Show state space of [Position].
  • v : Show state space of [Velocity].
  • a : Show state space of [Acceleration].
  • b : Draw bounds of state variables.
